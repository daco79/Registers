# Dictionnaire des domaines

## Domaines principaux

| Domaine | Description |
|---|---|
| Entreprises | Fiche juridique, RCS/RNE, établissements, dirigeants, finances, actes, BODACC, procédures, sanctions |
| Dirigeants | Personnes physiques ou morales exerçant des fonctions dans une entreprise |
| Bénéficiaires effectifs | Détention du capital, droits de vote, contrôle ; accès parfois soumis à habilitation |
| Comptes / finances | Ratios financiers structurés, comptes sociaux ou consolidés, éventuellement bilan complet |
| Documents | Actes, statuts, comptes PDF/XLSX via token |
| Immobilier | Parcelles cadastrales, propriétaires, occupants, ventes, bâtiments, DPE, copropriétés, permis, urbanisme |
| Urbanisme | PLU, PLUi, SCoT, SUP, zones, servitudes |
| Cartographie | Graphe de relations entre entreprises, dirigeants, bénéficiaires et entreprises liées |

## Familles de données sensibles

- Scoring financier et non financier : à appeler uniquement sur demande explicite.
- Bénéficiaires effectifs : peut nécessiter une habilitation.
- Coordonnées personnelles, emails, téléphones : disponibles seulement via certains champs supplémentaires.
- Personnes politiquement exposées et sanctions : à signaler clairement si actives.
